# マンション購入シミュレーター

マンション価格を動かすと、月々の家計と将来の資産（ローン残債・貯金・NISA）がその場で更新される、ブラウザ完結のシミュレーターです。

**▶ デモ:** https://yoichiro-ashibe-umu.github.io/-mansion-simulator/

## 特長

- **価格は常に主役** — マンション価格スライダーと収支サマリーは常時表示。頭金・金利・年収・生活費・NISA などは「詳細設定」に折りたたみ。
- **設定を相方に共有** — 「🔗 共有リンクをコピー」を押すと、現在の設定をすべて埋め込んだURLが生成されます。相手がそのURLを開くだけで、同じ設定で表示されます。
- **自動保存** — 同じ端末・ブラウザでは設定が `localStorage` に自動保存され、次回も復元されます。
- **ビルド不要** — 素のHTML / CSS / JavaScript のみ。フレームワーク・依存パッケージなし。チャートも自作SVGで軽量。

## 仕組み（データ保持・共有）

設定はすべて単一の状態オブジェクトに集約され、

1. 変更のたびに `localStorage` へ自動保存（`mansion-sim:v2`）
2. 「共有リンクをコピー」で、状態を JSON → Base64 化し URL の `#` 以降に埋め込み
3. 読み込み時の優先順位は **URLハッシュ > localStorage > 既定値**

共有リンクで開いた直後にURLはクリーンな状態に戻すため、アドレスバーが長いハッシュで埋まり続けることはありません。

## ローカルで動かす

```bash
# このフォルダで簡易サーバーを起動（例）
python3 -m http.server 4567
# → http://localhost:4567 を開く
```

`index.html` をダブルクリックして直接開いても動作します。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面の構造 |
| `styles.css` | デザインシステム（配色・タイポ・レイアウト） |
| `app.js` | 状態管理・計算・チャート描画・共有/保存 |

## 注意

手取りは額面の約78%、管理費・修繕積立金・諸費用は概算です。固定資産税・保険・ボーナス返済等は含みません。NISA資産は想定成長率での単純計算で、運用成果を保証するものではありません。これは概算ツールであり、投資・融資の助言ではありません。実際の数値は金融機関やファイナンシャルプランナーにご確認ください。
